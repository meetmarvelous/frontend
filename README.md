# Symphora Art

An AI art generation and prompt marketplace platform.

## Installation & Setup

### Prerequisites

- Node.js 20 or higher
- npm

### Environment Variables

Create a `.env` file in the root directory and configure the following variables:

```bash
#privy
NEXT_PUBLIC_PRIVY_CLIENT_ID=
NEXT_PUBLIC_PRIVY_APP_ID=

#thirdweb
NEXT_PUBLIC_THIRDWEB_CLIENT_ID=
THIRDWEB_SECRET_KEY=

#gemini
GEMINI_API_KEY=
```

### Install Dependencies

```bash
npm install

npm run dev
```

npm run devOpen [http://localhost:3000](http://localhost:3000) in your browser to see the result.
